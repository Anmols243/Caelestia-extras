# Wallpaper Studio

## Vision

Wallpaper Studio is a modern wallpaper manager for Quickshell.

Unlike launcher-based solutions, Wallpaper Studio provides a native graphical experience with thumbnails, search, favorites, collections, and multi-monitor support.

---

## MVP

- Grid of wallpapers
- Search
- Click to apply
- Remember last wallpaper
- Restore on login

---

## Version 1.1

- Categories
