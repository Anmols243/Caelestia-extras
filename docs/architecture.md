# Architecture

## Vision

Caelestia Extras is a collection of polished Quickshell applications that extend Caelestia and other Quickshell-based desktops.

Each application should be:

- Modular
- Reusable
- Keyboard-first
- Beautiful
- Fast

---

## Applications

- Wallpaper Studio
- Audio Center
- Clipboard Manager
- Theme Manager
- Bluetooth Center
- Widgets
