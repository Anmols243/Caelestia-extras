# Wallpaper Model

Every wallpaper, regardless of provider, follows the same schema.

```json
{
    "id": "",
    "title": "",
    "preview": "",
    "provider": "",
    "type": "",
    "favorite": false,
    "tags": [],
    "last_used": "",
    "metadata": {}
}
```

Providers may add extra metadata, but these fields are guaranteed.
