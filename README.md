# Caelestia Extras

A collection of polished Quickshell applications for Caelestia and other Quickshell-based desktops.

> Modern. Beautiful. Modular.

---

## Planned Applications

- 🖼 Wallpaper Studio
- 🎵 Audio Center
- 📋 Clipboard Manager
- 🎨 Theme Manager
- 🦷 Bluetooth Center
- 📊 System Monitor

---

## Goals

- Native Quickshell applications
- Smooth animations
- Keyboard-first workflow
- Beautiful UI
- High performance
- Easy to extend
